# 1037Market

欢迎来到 1037集市，校园内的二手交易平台，致力于为您提供一个便捷、安全、高效的二手交易体验。

## 功能特色

- **浏览商品**：浏览校园内其他用户发布的二手物品，快速找到心仪之物。

- **发布商品**：轻松发布您的二手物品，描述清楚商品信息，吸引潜在买家。

- **消息通知**：实时消息通知系统，便于买卖双方的沟通与交流。

- **搜索功能**：通过关键词搜索您感兴趣的商品，快速定位您所需物品。

- **校园认证**：支持使用校园认证，确保交易安全可靠。

## 使用技术

- **前端框架**: Vue
- **后端框架**: Gorilla
- **数据库**: MySQL
